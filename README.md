### To use script that directly pump MKS pressure gauge data to database, go to MKS_gauge
```bash
cd ./MKS_gauge/
```

### To use pyQt5 python GUI, go to pyqt5
```bash
cd ./pyqt5/
```

### To use python GUI, go to Python_UI
```bash
cd ./Python_UI/
```

### To use mySQL database and grafana, go to grafana
```bash
cd ./grafana/
```
